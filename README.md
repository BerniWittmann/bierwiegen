# 🍺 Bierwiegen

Eine Web-App für das Bier-Golf Wiegespiel — jetzt als GitHub Pages gehostet.

**🌐 Live:** `https://bierwiegen.bernhardwittmann.com`

## Was ist Bierwiegen?

Bierwiegen kombiniert Golf-Scoring mit Biertrinken. Jedes Loch hat ein Zielgewicht (Par). Die Differenz zwischen tatsächlich getrunkener Biermenge und Par ergibt die Punkte — je näher am Par, desto besser.

## Features

- ⚙️ **Setup** — Spieler verwalten, Löcher & Par-Gewichte festlegen
- 📝 **Eingabe** — Gewichte pro Loch & Spieler eintragen, Live-Delta-Anzeige
- 📊 **Scorecard** — Vollständige Tabelle aller Löcher & Deltas
- 🏆 **Rangliste** — Leaderboard mit Statistiken
- 💾 Daten werden automatisch im Browser gespeichert (localStorage)

## Lokale Nutzung

Einfach `index.html` im Browser öffnen — keine Installation nötig.

## Deployment

Automatisches Deployment via GitHub Actions auf GitHub Pages bei jedem Push auf `main`.
